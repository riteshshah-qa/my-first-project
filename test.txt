# 🔹 SCENARIO 1: Create Local Repo → Push to GitHub

## ✅ Test Case 1: Initialize Local Repository

* **Precondition:** Git installed
* **Steps:**

  1. Create folder `ritzen`
  2. Run `git init`
* **Expected Result:**

  * `.git` folder created
  * Repo initialized successfully

---

## ✅ Test Case 2: Add and Commit File

* **Steps:**

  1. Create `test.txt`
  2. Run `git add .`
  3. Run `git commit -m "first commit"`
* **Expected Result:**

  * File committed successfully
  * Commit ID generated

---

## ❌ Test Case 3: Commit Without Adding File

* **Steps:**

  1. Create file
  2. Run `git commit -m "test"`
* **Expected Result:**

  * Error: *nothing to commit*

---

## ❌ Test Case 4: Commit Without Message

* **Steps:**

  ```bash
  git commit
  ```
* **Expected Result:**

  * Editor opens OR error for missing message

---

## ✅ Test Case 5: Add Remote Repository

* **Steps:**

  ```bash
  git remote add origin <repo-url>
  ```
* **Expected Result:**

  * Remote added successfully

---

## ❌ Test Case 6: Add Duplicate Remote

* **Steps:**

  * Run same command again
* **Expected Result:**

  * Error: remote origin already exists

---

## ✅ Test Case 7: Push to Remote (First Time)

* **Steps:**

  ```bash
  git push -u origin main
  ```
* **Expected Result:**

  * Code pushed successfully
  * Upstream tracking set

---

## ❌ Test Case 8: Push Without Remote

* **Steps:**

  ```bash
  git push
  ```
* **Expected Result:**

  * Error: no remote configured
